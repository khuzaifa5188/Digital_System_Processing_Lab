## 🌟 Introduction

This repository contains MATLAB implementations for various DSP lab tasks, focusing on:

- Generation and analysis of discrete-time signals
- Frequency domain transformations (DTFT, FFT)
- Signal processing fundamentals
- Visualization of complex signals

## 🧪 Labs

### Lab 01: Generation and Analysis of Discrete-Time Signals
- **Task 1**: Basic Non-Periodic Sequences (Unit Sample, Unit Step, Unit Ramp)
- **Task 2**: Sawtooth and Square Waves
- **Task 3**: Complex Exponential Analysis
- **Task 4**: Sinusoidal Signal Analysis and Aliasing

📁 [lab01_task1.m](lab01_task1.m)

### Lab 02: Frequency Domain Analysis (DTFT and FFT)
- **Task 1**: Fourier Transform using FFT for finite sequences
- **Task 2**: DTFT plotting for exponential sequences
- **Task 3**: Frequency response from difference equations
- **Task 4**: Comparison of manual DTFT vs built-in `freqz`

📁 [lab02.m](lab02.m)

## 🛠 Requirements

- MATLAB R2018a or later
- Signal Processing Toolbox (for `freqz`, `sawtooth`, `square` functions)

## 🚀 Usage

1. Clone this repository:
   ```bash
   git clone https://github.com/khuzaifa5188/Digital_System_Processing_Lab.git
   ```

2. Open MATLAB and navigate to the repository folder.

3. Run the desired lab script:
   ```matlab
   lab01_task1  % For Lab 1
   lab02        % For Lab 2
   ```

4. View the generated plots and analyze the results.
